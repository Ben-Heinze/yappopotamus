* Components of MLR Models vs GLM

** Components of MLR Models

\[\mu\{Y | X_1, \ldots, X_p\} = \beta_0 + \beta_1 X_1 + \beta_2 X_2 + \cdots + \beta_p X_p\]
\[\text{Var}\{Y | X_1, \ldots, X_p\} = \sigma^2\]

** Components: GLM (binary Y)

- Systematic component (distribution for \(Y\)): [FILL IN]
- Linear predictor: [FILL IN]
- Link function: [FILL IN]

Models \(\mu\{Y | X\} = \pi\) as a linear function of X variables.

- \(\eta = \text{logit}(\pi) = \beta_0 + \beta_1 X_1 + \cdots + \beta_p X_p\)
- \(\pi = \text{logit}^{-1}(\eta) = \dfrac{e^\eta}{1 + e^\eta}\) (logistic function)

* Modeling Odds

\[\text{logit}(\pi) = \log\!\left(\frac{\pi}{1 - \pi}\right) = \beta_0 + \beta_1 X_1 + \cdots + \beta_p X_p\]

\Rightarrow Odds of a "success"/yes/Y = 1 for specific X: [FILL IN]

* Interpreting \(\beta_0\)

\[\text{logit}(\pi) = \log\!\left(\frac{\pi}{1 - \pi}\right) = \beta_0 + \beta_1 X_1 + \cdots + \beta_p X_p\]

\Rightarrow The odds of "success" (\(Y = 1\)) at \(X_1 = 0, \ldots, X_p = 0\): [FILL IN]

* Interpreting \(\beta_k\)

\[\text{logit}(\pi) = \log\!\left(\frac{\pi}{1 - \pi}\right) = \beta_0 + \beta_1 X_1 + \cdots + \beta_p X_p\]

The ratio of the odds for \(X_1 = A\) relative to \(X_1 = B\), while holding other variables fixed:

\[\frac{\omega_A}{\omega_B} = \frac{\exp\{\beta_0 + \beta_1 A + \cdots + \beta_p X_p\}}{\exp\{\beta_0 + \beta_1 B + \cdots + \beta_p X_p\}} = \exp\{\beta_1(A - B)\}\]

\Rightarrow For a 1-unit change in \(X_1\): [FILL IN]

* Maximum Likelihood Estimation

** How Did We Get \(\hat{\beta}\)?

Maximum likelihood (ML) estimation is used for GLMs.

- \(Y \sim \text{Bernoulli}(\pi)\); \(\text{logit}(\pi) = \beta_0 + \beta_1 X_1 + \cdots + \beta_p X_p = X\beta\)
- \(L(\beta \mid Y_1, \ldots, Y_n) = \displaystyle\prod_{i=1}^n \pi^{y_i}(1-\pi)^{1-y_i}\)
- \(L(\beta \mid Y_1, \ldots, Y_n) = \left(\dfrac{e^{X\beta}}{1 + e^{X\beta}}\right)^{\sum y_i} \left(1 - \dfrac{e^{X\beta}}{1 + e^{X\beta}}\right)^{n - \sum y_i}\)

** Properties of ML Estimators

If the model is reasonable and the sample size is large enough:

1. MLEs are unbiased: \(E(\hat{\beta}_j) = \beta_j\)
2. A formula exists for finding \(\text{SE}(\hat{\beta}_j)\)
3. \(\text{SE}(\hat{\beta}_j)\) is smaller for MLE than any other nearly unbiased estimator
4. The distribution of \(\hat{\beta}_j\) is approximately normal

* Inference

** Wald-Based Tests for Single \(\beta_j = 0\)

- \(H_0: \beta_j = 0\) given \(X_{-j}\) — no logit-linear association between \(X_j\) and probability of "success", given other variables in the model
- \(H_a: \beta_j \neq 0\) given \(X_{-j}\) — some logit-linear association between \(X_j\) and probability of "success"

** Drop in Deviance (Likelihood Ratio Test)

Allow full vs. reduced model comparisons in GLMs using likelihood ratio tests (LRT).

- TS: \(\text{LRT} = 2\log(L_{\text{MAX,full}}) - 2\log(L_{\text{MAX,reduced}})\)
  - deviance \(= \text{constant} - 2\log(L_{\text{MAX}})\)
  - \(\text{LRT} = \text{deviance}_{\text{reduced}} - \text{deviance}_{\text{full}}\)
- Under \(H_0\) ("reduced model is adequate"): \(\text{LRT} \sim \chi^2_{df_{\text{red}} - df_{\text{full}}}\)
- LRT in GLM is analogous to ESS F-tests in MLR
