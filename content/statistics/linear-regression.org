* Linear Regression in R

Simple linear regression using R's built-in =lm()= function. We generate noisy data
from a known line (slope 2.5) and recover the coefficients.

#+NAME: linear-regression
#+begin_src R :results output
set.seed(42)
x <- 1:20
y <- 2.5 * x + rnorm(20, sd = 4)

model <- lm(y ~ x)

cat("--- Coefficients ---\n")
print(coef(model))

cat("\nR-squared:", round(summary(model)$r.squared, 4), "\n")
cat("Residual SE:", round(summary(model)$sigma, 4), "\n")
#+end_src
