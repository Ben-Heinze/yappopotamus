* Manual Org Table

Type your table with =|= separators. In Emacs, press ~TAB~ inside a table
cell to auto-align all columns and jump to the next cell. Press ~RET~ to
move to the next row. Type =|-= and press ~TAB~ to insert a horizontal rule.

| Name    | Subject     | Score |
|---------+-------------+-------|
| Alice   | Mathematics |    95 |
| Bob     | Physics     |    82 |
| Carol   | Chemistry   |    91 |

** Table with a caption

Add ~#+CAPTION:~ on the line directly above the table. It appears as a
visible caption below the table in the exported HTML.

#+CAPTION: Exam results — Spring 2026
| Name    | Subject     | Score | Grade |
|---------+-------------+-------+-------|
| Alice   | Mathematics |    95 | A     |
| Bob     | Physics     |    82 | B     |
| Carol   | Chemistry   |    91 | A     |

* Table Generated from Python

Use ~:results value table~ to return data from a code block as an HTML table.
The block must use ~return~ (org-babel wraps Python in a function behind the scenes).
~:colnames~ sets the header row.

#+begin_src python :results value table :colnames '("n" "n²" "n³" "n⁴")
return [(i, i**2, i**3, i**4) for i in range(1, 7)]
#+end_src

* Table Generated from R

R data frames export cleanly with ~:results value table :colnames yes~.

#+begin_src R :results value table :colnames yes
data.frame(
  n      = 1:6,
  square = (1:6)^2,
  cube   = (1:6)^3,
  fourth = (1:6)^4
)
#+end_src
