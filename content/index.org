* How snippets work

Each subject folder contains small =.org= files — one concept per file. The folder's =index.org= pulls them together with =#+INCLUDE=:

#+begin_example
* Recursion

#+INCLUDE: "fibonacci.org" :minlevel 2
#+end_example

The =:minlevel 2= makes the snippet's top-level heading nest one level
deeper, so it appears as a subsection of the including page.

A snippet can be included in multiple pages — edit it once, and every
page that includes it updates on the next build.
